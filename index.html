Hysummit

Aug 2023 - Mar 2024 PERSONAL PROJECT GROUP PROJECT

Hysummit is a Minecraft server which uses server side mods to add custom gameplay mechanics without requiring players to download mods.
I created a server side Minecraft mod that implements a multi-server dungeon instancing system using Redis pub/sub for communicating between servers and SQLite for persistence using Java, with a lobby and party system enabling players to form private or public groups and launch new dungeon instances on remote servers once minimum party size was met.

Languages & Libraries: Java, PaperMC, Redis, SQLite
