Chimp

Chimp is a 3D game engine with networking, lighting & shadow support, music & positional sound effects, editor, ECS supporting serialisation, and ref-counted resource management. It was originally started as a university project and later continued as a passion project.

Main Features

Graphics

  • Render textured sprites and 3D models
  • Basic lighting (4 spotlights, 4 point lights, 4 directional lights)
  • Shadow mapping (including omnidirectional shadow mapping)
  • Supports custom user-written shaders (no lighting in custom shaders though)
  • OpenGL backend

Networking

  • Packets can be sent as binary data over TCP protocol
  • Chimp can host a server or connect to another server
  • Packets can be listened for using events
  • Packets support callbacks
  • Clients communicate via server (not peer-to-peer)
  • Networking runs on dedicated thread; events & callbacks run on main thread
  • ENet backend

Entities

  • ECS wrapping FLECS
  • Entity hierarchy with parent/child transforms
  • JSON de/serialisation
  • Custom reflection system for serialisation and editor integration

Resource Management

  • Automatic unload of unused resources
  • No reload of shared resources when switching scenes

Audio

  • Sound effect system with pitch/volume variation
  • Positional audio
  • Music with cross-fade
  • OpenAL backend

Editor

  • Optional; full code workflow supported
  • Scene view with live updates
  • Entity creation/removal
  • Entity inspector for components
  • Asset manager for browsing data
  • Asset importer (WIP, editor-only)
